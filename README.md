# Naive-Bayes
```
Celem jest napisanie programu, który dla jako argumenty otrzymuje zbiór treningowy oraz zbiór testowy. Na podstawie zbioru treningowego, program ma za zadanie zaklasyfikować przypadki ze zbioru testowego korzystając z naiwnego klasyfikatora Bayesa. Należy również zapewnić możliwość ręcznego podawania danych do klasyfikacji. Należy zastosować wygładzanie Laplace’a.

UWAGA! Program powinien działać poprawnie dla dowolnej liczby atrybutów.
```
